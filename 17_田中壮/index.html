パズルモンスターズ

GitHub

ブロックを動かして敵を討伐せよ！

パズルモンスターズは、ブラウザですぐに遊べる新感覚パズルRPGです。
同じ色のブロックを揃えて敵にダメージを与え、迫りくる強敵たちを打ち倒そう！

🎮 ゲームをプレイする
🎮 ゲームURL https://SoTanaka0409.github.io/javawebgame/game.html
🐙 GitHub URL https://github.com/SoTanaka0409/javawebgame

💡 開発の工夫・こだわった点

🔥💧🌳

属性の相性システム

敵モンスターには属性が設定されており、消すブロックの色によってダメージが変動します。状況に合わせて消す色を変える戦略性を重視しました。

✨⚔️

MPを使ったスキルシステム

パズルを消してMPを貯めると、「クロス斬り」や「ヒール」などの強力なスキルが発動可能に。ピンチをチャンスに変える爽快感を作り込んでいます。

🎵🔊

Web Audio APIによるサウンド

外部の音声ファイルに頼らず、ブラウザ標準のWeb Audio APIを活用。コンボが続くたびに音程が上がるなど、リッチなサウンド体験を実現しました。

🎨💻

モダンなUIとアニメーション

Tailwind CSSを使用し、HPバーの滑らかなアニメーションやボタンのエフェクトなど、プレイヤーが触っていて心地よいUIを目指しました。
